│ Create New Knowledge Base                           [×] │

│  └───────────────────────────────────────────────┘     │

│                                                         │
│  Description (optional)                                 │
│  ┌───────────────────────────────────────────────┐     │
│  │ Project documentation for...                 │     │
│  │                                               │     │
│  └───────────────────────────────────────────────┘     │
│                                                         │

│  │ • my_new_project_vgrag_relations             │     │
│  │ • my_new_project_vgrag_passages              │     │

│                                                         │
│                            [Cancel]  [Create]           │
└─────────────────────────────────────────────────────────┘

验证规则：
✅ 名称不能为空
✅ 只能包含 [a-zA-Z0-9_-]
✅ 长度 1-50 字符
✅ 不能与现有知识库重名
❌ 实时显示错误提示

    🎨 视觉设计规范¶

    颜色方案¶

    知识库选择器：
├─ 背景: bg-slate-50
├─ 边框: border-slate-200
├─ 选中项: bg-blue-50, text-blue-700
├─ 悬停: bg-slate-100
└─ 图标: text-slate-500

统计信息：
├─ 背景: bg-accent/20
├─ 数字: text-slate-700, font-semibold
├─ 图标: 使用对应颜色
│   ├─ 实体: text-blue-500
│   ├─ 关系: text-green-500
│   └─ 文档: text-purple-500
└─ 边框: border-slate-200

按钮：
├─ 主按钮: bg-blue-600, hover:bg-blue-700
├─ 次按钮: border-slate-300, hover:bg-slate-50
└─ 危险操作: bg-red-600, hover:bg-red-700

    🔄 交互流程¶

    场景 1: 首次使用（默认知识库）¶

    用户打开导入对话框
  ↓
默认选中 "default" 知识库
  ↓
显示提示："正在导入到 default 知识库"
  ↓
用户可以选择切换或继续
  ↓
导入成功，显示详细统计

    场景 2: 选择现有知识库¶

    用户打开导入对话框
  ↓
点击知识库选择器
  ↓
下拉菜单展开，显示所有知识库
  ↓
用户选择 "my_project"
  ↓
统计信息自动更新，显示 my_project 的数据
  ↓
用户上传文档
  ↓
数据导入到 my_project

    场景 3: 创建新知识库¶

    用户打开导入对话框
  ↓
点击知识库选择器
  ↓
点击 "➕ Create New Knowledge Base"
  ↓
弹出创建对话框
  ↓
输入名称 "research_papers"
  ↓
实时验证（显示绿色勾号 ✅）
  ↓
预览将创建的 collections
  ↓
点击 "Create"
  ↓
创建成功，自动选中新知识库
  ↓
用户可以立即开始导入

    🎯 可访问性 (A11y)¶

    键盘导航¶

    Tab           → 在控件间切换焦点
Enter         → 确认/提交
Escape        → 关闭对话框
Space         → 切换开关/选中
Arrow Up/Down → 在下拉列表中导航
